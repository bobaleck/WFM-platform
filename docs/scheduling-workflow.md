# Workflow графиков

Планирование выполняется по цепочке:

1. Создать сотрудников или загрузить реестр XLSX.
2. Назначить сотрудникам проекты, команды и глобальные навыки.
3. Создать очереди и требуемые навыки очередей.
4. Создать смены как шаблоны рабочих интервалов.
5. Загрузить нагрузку вручную через XLSX/CSV.
6. Рассчитать потребность.
7. Сформировать черновик графика.
8. Проверить покрытие и рекомендации.
9. Подтвердить и опубликовать график.

Отсутствия выбирают сотрудника из списка активного проекта и должны блокировать назначение на период отсутствия.
