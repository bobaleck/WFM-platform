# Production checklist

- [ ] Пароль администратора изменён.
- [ ] `JWT_SECRET` заменён.
- [ ] `AUTH_ENABLED=true`.
- [ ] `.env` имеет права `600`.
- [ ] PostgreSQL не открыт наружу.
- [ ] Redis не открыт наружу.
- [ ] Backend и scheduler доступны только локально/proxy.
- [ ] Включён TLS на reverse proxy.
- [ ] Настроен backup.
- [ ] Настроена проверка восстановления.
- [ ] Настроен мониторинг диска.
- [ ] Настроен мониторинг health endpoints.
- [ ] Настроена ротация логов.
- [ ] Настроена политика пользователей и ролей.
- [ ] Проверен audit log.
- [ ] Проверены экспорты.
- [ ] Проверены персональные данные.
- [ ] Проверены лицензии open-source компонентов.
- [ ] Проверена стратегия обновлений.
- [ ] Подготовлен план подключения внешнего источника данных на будущих этапах.
