# WFM backend module

Здесь будут находиться доменные сервисы WFM:

- сотрудники, команды и навыки;
- очереди и интервальная нагрузка;
- расчёт потребности в операторах;
- смены, назначения и отсутствия;
- KPI и отчёты;
- adapter layer для Naumen.

На Этапе 1 создан только skeleton без бизнес-логики и без интеграции с Naumen.
