# Дизайн-система «Телесейлз Сервис»

Дата обновления: 2026-05-27.

## Позиционирование

Интерфейс — внутренняя enterprise WFM/BI-платформа для контактного центра «Телесейлз Сервис». Основная аудитория: руководители, супервизоры, аналитики и администраторы операционного контура.

Ключевой принцип: рабочий инструмент высокой информационной плотности, а не демонстрационный SaaS-макет.

## Визуальный принцип

- строгий;
- деловой;
- компактный;
- спокойный;
- высокая читаемость;
- минимум декоративности;
- панели и таблицы вместо крупных «плавающих» карточек.

## Типографика

- body: 14px, weight 400;
- подписи: 13px, weight 400-500;
- меню: 14px, weight 500;
- заголовок страницы: 24-26px, weight 600;
- заголовок панели: 18px, weight 600;
- KPI label: 13px, weight 500;
- KPI value: 26-30px, weight 600;
- таблицы: 13px, weight 400-500.

Жирность 700/800/900 в интерфейсе не используется как базовый стиль.

## Layout

Sidebar: 248px, белый фон, тонкая граница, компактные пункты 38px. Активный пункт — светло-синий фон, синий текст и тонкий левый индикатор.

Header: 56-64px, белый фон, нижняя граница, компактные status pills. Имя пользователя обычным текстом.

Карточки и панели: radius 12px, border, минимальная тень.

Таблицы: header `#F9FAFB`, компактные строки, hover `#F9FAFB`, числовые колонки выравниваются вправо.

## Ограничения

Чужие логотипы не используются. Внешние картинки и шрифты не скачиваются. Реальные клиентские бренды в демо-данных не используются.
