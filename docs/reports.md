# Отчёты WFM

Отчёты показывают данные после прохождения основного workflow: загрузка нагрузки, расчёт потребности, формирование графиков и загрузка факта, если он доступен.

Разделы:

- Руководитель: сотрудники, очереди, графики, покрытие, SLA и AHT.
- Операции: нагрузка по очередям.
- Покрытие: потребность против плана.
- План/факт: плановые часы против фактических, если факт загружен.
- SLA: показатели SL/SLA по очередям.
- Экспорты: журнал выгрузок.

Если данных нет, интерфейс должен объяснять, какие исходные данные нужно загрузить. Выгрузки выполняются через авторизованный запрос.
